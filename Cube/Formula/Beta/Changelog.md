## [7.18.1321-beta.66] - 2026-07-15

### Added
- 操作面板里的“自定义复原状态”改为在面板内部就地展开，不再弹出遮挡整屏的窗口。
- “导出公式”（公式制作）改为面板内就地展开。
- “公式库导出”也改为面板内就地展开，与“导出公式”统一。
- 记忆模式的“规划记忆”改为面板内就地展开，并支持再次点击触发按钮收起。
- 记忆模式“规划记忆”内的“编辑公式”改为面板内就地展开（可“收起”）。
- 危险操作新增带警示色的统一确认提示，取代浏览器原生确认框：包括删除公式组，以及记忆模式的删除/替换/覆盖。
- “添加公式到公式组”对话框新增图标与当前组名强调。
- “添加公式到公式组”的“其他组”列表支持展开/收起（不再点击即整块替换）。

### Changed
- 操作面板整体去框化简：移除分组边框、背景、圆角与标题，面板内不再有分割线，仅靠间距区分区块。
- “显示公式”开关实时显示已导入公式数量（共 N 个）与当前公式名称，取代底部“已导入 N 个公式”提示。
- “添加公式到公式组”对话框升级为卡片式，说明文案改为更直白的“选择这组公式的保存位置”，取消按钮改为轻量样式。
- “其他组”列表不再重复显示当前所在组。
- 横屏操作面板的开关把手改为贴右缘的常驻竖条（不再悬浮覆盖内容），右侧新增渐隐装饰带，开合更顺滑。
- 操作面板及各区块底色统一，头部在竖屏/横屏下采用与布局一致的渐变。
- 内联展开内容（自定义状态/导出/规划/编辑）采用统一的卡片样式与展开动画。
- “自定义复原状态”面板内的魔方预览改为自适应高度（移动端更大更清晰）。
- “导出公式”面板布局优化：文本框可拖拽调整高度，操作按钮自动换行。
- 统一确认提示与“添加公式到公式组”对话框在窄屏（手机）下改为从底部滑出的卡片式弹窗。
- 打开“自定义复原状态”面板时，操作面板的 Tab 键盘焦点切换保持正常（不再被旧遮罩判断跳过）。
- 适配系统“减弱动态效果”偏好：开启时不再播放展开/提示的过渡动画。

### Removed
- 移除底部“已导入 N 个公式”状态文字（由“显示公式”开关状态取代）。
- 移除导入时的瞬时状态文字（“正在读取…”“读取失败…”“已导入 N 个、跳过 M 条”等），改为开关状态统一反馈。
- 移除“其他组”按钮点击后整块替换的交互，改为可展开/收起列表。
- 移除“自定义复原状态”“导出公式”等弹窗点击空白处关闭的旧逻辑（现通过“收起/取消/关闭”按钮或再次点击触发按钮关闭）。

### Fixed
- 修复打开“自定义复原状态”时操作面板 Tab 键盘焦点切换失效的问题。

## [7.16.1168-beta.64] - 2026-07-11

### Added
- 复原检测新增两种模式（单选框切换，默认第二种）：过程检测与最终状态检测。
- 记忆答案界面显示「今日进度 N%」。
- 同日重复按间隔公式数量排队：清晰 4、犹豫 3、模糊 2、遗忘 1。

### Changed
- 学习/训练/记忆三模式共用同一公式文本框与同一套导入逻辑。
- 文本框仅接受公式输入，题目初始状态由公式逆算得到（内部计算，不对外展示）。
- 任意面连续转四下恢复当前题目初始状态（训练全程、记忆显示与隐藏答案时均生效，答案界面保持不动）。
- 最终状态检测按当前公式组复原目标比较，切换组立即更新目标。
- 「显示公式」改为显示导入的标准化原公式。
- 同日重复改为按公式数量间隔；仅当天首次评分更新排程，后续评分只更新当日进度与重复队列。

### Removed
- 移除公式制作模块独立的 M/E/S 展开逻辑，改用全局统一逻辑。
- 移除以「状态」为中心的旧文本解析入口（状态文本、状态导入等）。
- 移除当日重复中的时间等待与定时唤醒逻辑。

### Fixed
- 修复公式制作录制 M/E/S 被错误换算为展开动作的问题，录制与显示保留原始 M/E/S 写法。
- 修复 M/E/S 在过程检测中与普通动作不一致的问题。
- 修复记忆模式转四下触发自动重试、答案界面状态更新不一致的问题。
